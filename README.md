# CBED Simulation

Generate realistic-looking CBED images from crystal definition and orientation.

Diffraction spot positions are generated with kinematic or dynamical
simulations, and images are built using compositing. Image properties (like
shot noise, saturation, texture, ...) can be influenced using a small number of
parameters.

Basic usage in [make_frame.ipynb](https://github.com/matbryan52/cbed-simulation/blob/main/make_frame.ipynb).
